HTML : 1 Basics

This will be a basic introduction on XHTML. By the end of this, you will be able to produce a simple webpage.

• A basic HTML document consists of having a <html>, <head> and <body>. The <html> is what makes your document a html. The <head> contains the title of the page and other information that is not displayed. <body> is where you write your content. Finally you need to close off these tags by writing '/' in front of the word of the tag (eg. </html> ).

• Tags always have a start and end to them. They also provides instructions on the structuring of the webpage.

• Tags are able to be nested within other tags: <div><p>This <em>paragraph<em> is nested</p></div>

Here are some tags which are most commonly used

Tag Explanation Appearance
<p> Using to write paragraphs
<em> Emphasises text Example
<strong> Marks important text Example
<div> For creating blocks of content
<br> Line breaking
<h1, h2, h3> Headers
<a href= " "> Inserting links Clickable
<span> Enclosing short lines of content
<img src= " "> Inserting images

• By using class and id attributes, you can group tags toether which makes creating the presentation much easier. It also is important to have these for when you write up your css.

• There are special characters and symbols which you cannot just type into your HTML. You must use their HTML number. For example: < is written as &#60;
For a list of all the symbols, check out this link!

• Lastly you will need to put a DTD (Document Type Definition) declaration at the beginning of your webpage to make your document XML acceptable.
As this webpage was written with XHTML 1.0 Transitional, this is the DTD code for it:

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">

Hopefully this lesson will have taught you the basics of HTML. Our next lesson will go into more detail of how each tags work.
