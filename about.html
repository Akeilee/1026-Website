About: HTML & CSS

The World Wide Web was created by Tim Berners-Lee in 1989. Tim Berners-Lee was working at CERN when he came up with the idea of the Web. After, he invented HTML and wrote his first version of it in 1993.

HTML otherwise known as Hypertext Markup Language is a markup language and this is the main language for producing webpages. HTML is used to create the base format of a webpage.It contains the content and structural semantics of a page.
Originally HTML was simple and used to handle text-based documents. However as using the web became more popular, HTML had to evolve to keep up with the demands of new features. HTML 2.0 to 3.2(1996) enabled images, forms and structuring of the page. In 1999 HTML 4.01 became the last complete standard version of HTML. It had support for CSS, JavaScript and other multimedia objects.
Eventually XHTML came to development in 2000. It was a rewrite of HTML using XML languages. XHTML is very similar to HTML but its rules are more strict. Most webpages are written in either HTML 4.01 or XHTML 1.0.
In more recent times HTML5 has slowly started to take over HTML 4.01 and XHTML. HTML5 was created in collaboration with W3C and WHATWG. Using this language would reduce the need for browser plugins, improve the responsiveness of the webpage on different devices and overall improve the semantic organisation. Document Object Models (DOM) and APIs are now also part of the specification.

CSS or Cascading Style Sheets is used in conjunction with HTML. It is a style sheet language and it creates the layout and presentation style of the HTML document. By using CSS, there is more flexibility and control in the webpage design. It also recduces the need of repeating code throughout multiple webpages. Another useful feature is that you are able to change the formatting for different devices by using @media rules and also be able to add animation and transformation effects.
Hakon Wium Lie created the first draft of CSS in 1994 and within two years, the first release of CSS Level 1 was made. This release supported many different formatting such as tables, images, colours, etc. Not long after, CSS 2 came out which supported z-indexes and other media types. In 2011 came CSS 3 which is the most current styling sheet.
